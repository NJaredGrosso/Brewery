Aqui pueden ver el codigo de cada funcion

IF ////////////////////////////////////////////////////

Aqui el if solo comprobará que el número que ingreses sea distinto a cero y te dirá cual es

let a = prompt ("ingrese un número distinto de 0");
if(a != 0){
    alert ("Número ingresado: " + a);
}else{
    alert ( "Valor no valido" );
}


FOR ////////////////////////////////////////////////////

En este caso con la funcion For hara un bucle en el que sumará una cuenta cada vez que se repita el loop hasta llegar a 10

for ( let b = 0; b < 11; b++ ) {

    alert ( "La cuenta va en: " + b )

}


WHILE ////////////////////////////////////////////////////

En este caso te solisitara que ingreses un usuario y siempre que el usuario no esté en la lista de baneados te saludará, en caso de estar baneado te lo informará, si escribes ESC fializará el codigo

let usuario = prompt ( "Ingrese usuario" );
let baneado = "juan";

while ( usuario != baneado ){
    alert ( "Bienvenido " + usuario );
    usuario = prompt ( "Ingrese otro usuario" )
    if ( usuario = ESC ){
        break;
    }
}

alert("Usuario no permitido");
